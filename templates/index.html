About the Project

CarSpec AI is a computer vision-based vehicle multi-attribute recognition system, trained using the CompCars dataset (130k+ images). The project implements three modeling approaches: Naive baseline, Classical ML (Random Forest + interpretable features), Deep Learning (MobileNetV2 multi-task learning), and provides better performance and interpretability than single-task models through multi-task joint learning and interpretable visual features.
